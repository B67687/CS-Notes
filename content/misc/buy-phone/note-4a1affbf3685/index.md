### 一、处理器

手机处理器（也称移动SoC，System on Chip，系统级芯片）是智能手机的核心组件，负责执行计算、处理数据、运行操作系统等任务。

#### 核心构成：
- **CPU（中央处理器）**：执行应用程序和用户指令。
- **GPU（图形处理器）**：处理图像渲染，对游戏和视频播放至关重要。
- **NPU（神经网络处理器）**：加速AI任务，如图像识别、语音处理。
- **ISP（图像信号处理器）**：处理摄像头捕获的图像信号。
- **DSP（数字信号处理器）**：优化音频和视频处理。
- **Modem（调制解调器）**：处理移动网络连接（如4G/5G）。

#### 主要制造商：
- 高通（Qualcomm）：Snapdragon 系列  
- 苹果（Apple）：A 系列芯片（用于 iPhone）  
- 华为（Huawei）：麒麟（Kirin）系列  
- 三星（Samsung）：Exynos 系列  
- 联发科（MediaTek）：天玑（Dimensity）系列  
- 小米（Xiaomi）：澎湃（Surge）系列  

#### 制程工艺：
- 越先进的制程（如5nm、4nm、3nm），性能越强，功耗越低。
- 当前高端芯片普遍采用3–5nm工艺。

#### 性能指标：
- **核心数**：主流手机多为8核（大核+小核组合），并非越多越好，能效调度更重要。
- **主频**：CPU时钟频率，越高理论性能越强。
- **能效比**：性能与功耗的比值，高能效比意味着更省电。

#### 发展趋势：
- **集成度提升**：更多功能（如5G基带、AI引擎）集成到单芯片。
- **AI加速**：NPU成为标配，支持本地AI应用。
- **5G支持**：已成为中高端芯片的标配。

### 二、内存和存储

#### 内存（RAM）：
- **作用**：运行应用程序，内存越大，可同时运行的App越多，系统更流畅，但功耗也略高。
- **建议配置**：
  - 3–4GB：基础使用（已不推荐）
  - 6GB：起步线
  - 8GB：日常使用足够
  - 12GB：适合大型游戏或多任务
  - 16GB+：重度用户或未来-proof 需求

> **内存扩展功能**（虚拟内存）：
> - 建议：≤6GB 开启（如+2/3/4GB），≥8GB 关闭。
> - 缺点：占用存储空间，可能引起卡顿。

#### 存储（ROM）：
- **作用**：安装应用、存储照片、视频、文件等。
- **建议容量**：
  - 32–64GB：仅轻度使用（需依赖云或存储卡）
  - 128GB：日常使用起点
  - 256GB / 512GB：推荐主流选择
  - 1TB / 2TB：适合摄影师、视频创作者

> **提示**：低于128GB建议选择带 **microSD 卡槽** 的机型，便于扩展。

#### 常见组合参考：
- 3GB + 32GB：仅能安装小型App  
- 4GB + 64GB：轻度日常使用  
- 6GB + 128GB：正常日常使用  
- 12GB + 256GB：游戏+拍照无压力  
- 16GB + 512GB：多应用+高清视频  
- 24GB + 1TB：专业级内容创作（支持8K视频录制与编辑）

### 三、屏幕

#### 尺寸：
- 主流约 **6.5 英寸**（对角线长度，单位：英寸）

#### 材质：
- **LCD**：成本低，护眼性好，但对比度低、更厚重。
- **OLED**：色彩鲜艳、对比度高、更薄，但存在“烧屏”风险（长期显示静态图像可能导致残影）。

> 建议：避免长时间停留在同一界面（如导航、状态栏），以延长OLED寿命。

#### 分辨率：
- 低端：720P（HD） / 480P  
- 主流：1080P（FHD）  
- 高端：2K（1440P） / 2.5K（1220P+） / 4K（极少见）

> 分辨率越高，显示越细腻，颗粒感越弱。

#### 刷新率：
- 60Hz：基础  
- 90Hz / 120Hz：主流高刷  
- 144Hz / 165Hz / 240Hz：游戏向高刷

> ≥90Hz 即可称为“高刷屏”，视觉更流畅。

#### 屏幕厂商推荐：
- 三星（Samsung Display）  
- 京东方（BOE）  
- 华星光电（CSOT）

### 四、拍照摄像

#### 关键因素（不止看像素）：
- 传感器尺寸（越大越好）  
- 镜头素质  
- ISP 图像处理能力  
- 软件算法调校  
- 长焦/超广角能力  
- 视频录制规格

#### 像素参考：
- ＜100万：低  
- 200–300万：中  
- 500–600万：中高  
- 800万+：高  
- 现代手机：4800万 / 5000万 / 6400万 / 1亿 / 2亿像素（默认输出多为1200万，超高像素用于裁切）

#### 优质传感器（部分）：
- 索尼：LYT-900 / IMX989 / IMX920 / IMX866 / IMX800 / IMX700  
- 豪威（OV）：OV50K / OV64B / OV50H / OV48C

#### 变焦类型：
- **光学变焦**：无损画质（依赖长焦镜头）  
- **数码变焦**：有损放大

#### HDR（高动态范围）：
- 提升亮部/暗部细节，色彩更丰富，优于普通SDR。

#### 常见照片格式：
| 格式 | 特点 |
|------|------|
| **JPG** | 有损压缩，兼容性好，文件小 |
| **PNG** | 无损压缩，支持透明，文件较大 |
| **HEIF** | 高效压缩（iOS默认），比JPG小50%，支持10bit/HDR |
| **GIF** | 256色，支持简单动画 |
| **RAW** | 无压缩原始数据，后期空间大，文件极大 |

#### 视频拍摄：

**分辨率与码率建议**：

| 分辨率 | 像素 | 推荐码率 | 说明 |
|--------|------|--------|------|
| 480P | 0.3MP | 1–5 Mbps | 标清，占用小 |
| 720P | 0.9MP | 6–10 Mbps | 高清，短视频常用 |
| 1080P | 2.1MP | 15–20 Mbps | 全高清，主流 |
| 4K | 8.3MP | 25–40 Mbps | 超高清，需大存储 |
| 8K | 33MP | 30–500 Mbps | 极高要求，耗电快 |

> **帧率（FPS）**：15 / 24 / 30 / 60 / 120 / 240（越高越流畅，慢动作需高帧率）  
> **码率**：越高越清晰，但文件越大。10 Mbps ≈ 1.25 MB/s

### 五、充电续航

#### 电池容量：
- ＜3500mAh：续航弱  
- 3800–4500mAh：中等  
- 4800–6500mAh+：强续航

> 建议选择 ≥4500mAh 机型。

#### 充电功率：
- 低功率：5W / 10W / 18W  
- 主流快充：30W / 50W / 65W / 67W  
- 高端快充：80W / 100W / 120W / 150W / 180W / 240W  
- 超快充：320W（实验室/极少数机型）

#### 无线充电：
- 支持 15W / 30W / 50W 无线充为佳

#### 无线反向充电：
- 可为耳机/手表充电，功率通常 5W / 10W

> **建议**：续航优先于快充，大电池更实用。

### 七、数据接口

#### 接口类型：
- **USB-C**：主流，正反插，支持快充/高速传输（**推荐**）  
- **Lightning**：iPhone 5–14 专用（iPhone 15 起改用 USB-C）  
- **Micro-USB**：老旧，仅10W充电  
- **Mini-USB**：极少见

#### 传输速度：
- USB 2.0：480 Mbps（≈60 MB/s）  
- USB 3.0/3.1/3.2：5–10 Gbps（≈625–1250 MB/s）

#### OTG（On-The-Go）：
- 支持手机直连U盘、键盘、鼠标等外设。

#### 耳机孔：
- 3.5mm 接口仍实用，但旗舰机逐步取消。

### 九、其他参数

#### 双卡双待：
- 多号码用户建议选择双卡槽机型。

#### 网络速度（理论峰值）：
- 2G：≈35 KB/s  
- 3G：≈150 KB/s  
- 4G：≈12 MB/s  
- 5G：≈100–120 MB/s  
- 6G（未来）：≈1.25 GB/s

#### 机身设计：
- **材质**：陶瓷 > 金属 > 玻璃 > 塑料  
- **重量**：
  - ＜150g：轻  
  - 150–180g：适中  
  - ＞200g：偏重  
- **厚度**：
  - ＜7mm：薄  
  - 7–8mm：正常  
  - ＞9mm：厚

#### 震动马达：
- **X轴线性马达** > Y轴线性马达 > 转子马达（体验依次下降）

#### 实用功能：
- **红外遥控**：可控制家电  
- **NFC**：支持公交卡、门禁模拟、支付  
- **蓝牙**：连接耳机/音箱  
- **WLAN**：Wi-Fi 6/6E/7 更佳

#### 音频：
- **麦克风**：双麦起步，高端机三麦/四麦（降噪更好）  
- **扬声器**：立体声 + 杜比/DTS/Hi-Res 认证为佳

#### 文件格式支持：
- **音频**：WAV（无损）> AAC/MP3（标准）> AMR（差）  
- **图片**：RAW > PNG > JPG > GIF  
- **视频**：AVI（无压缩）> MP4（标准）> 3GP（差）  
- **文档**：Word > TXT > RAR（非文档格式）

#### 传感器：
- 光线、距离、重力、加速度、磁场、陀螺仪等，提升交互体验。

#### 解锁方式：
- **指纹**：超声波 > 光学 > 电容（屏下体验依次下降）  
- **面部**：3D结构光（安全）> 2D图像（一般）

> 所有生物识别需配合密码使用。

#### 防水防尘：
- **IP68 / IP69**：高端机标配，支持短时浸水。

#### 耐摔性：
- 建议选择通过军规测试（如MIL-STD-810H）或强化玻璃（如Victus）的机型，延长使用寿命。

---

> **总结**：选购手机应综合性能、续航、屏幕、影像、系统体验与个人需求，旗舰机型在长期使用中更具优势。
