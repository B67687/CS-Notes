🔧 Performance Settings
- Enable Efficiency Mode
- Use edge://discards
- Enable Sleeping Tabs
- Set Resource Controls (RAM limiter) under System and performance
- Use Edge Task Manager (Shift + Esc) to monitor tab/extension usage
- Turn on Performance Detector to flag heavy tabs

🧼 Cleanup & Memory Management
- Clear download history (safe to remove)
- Trim browsing history (e.g. older than 30–90 days)
- Optionally clear cached images/files (318MB is fine unless buggy)
- Avoid clearing Autofill form data or Passwords
- Reset Edge settings if things feel bloated

🧠 Autofill & Sync Tips
- Keep Favorites, Passwords, Personal Info, Settings, and Feature Usage sync ON
- Turn OFF sync for Apps, Extensions, and optionally History
- Autofill works without browsing history — just don’t clear form data

🔐 Site Permissions
- Safe to clear if permissions were given accidentally
- Manage individually via Settings > Cookies and site permissions > All sites

🧩 Extensions & Background Control 
- Fully remove unused extensions (not just disable)
- Turn off Startup Boost and Background apps after close
- Prefer built-in features over heavy extensions

🗂️ Tab Management
- Use tab groups or vertical tabs to reduce clutter
- Try OneTab to consolidate open tabs
- Monitor usage via Browser Essentials sidebar

🧨 Advanced Tweaks
- Set Edge to High Performance in Windows Graphics Settings
- Flush DNS cache via edge://net-internals/#dns
- Disable Preload New Tab Page in settings
- Use Focused layout for New Tab Page
- Turn off Windows animations for snappier UI
