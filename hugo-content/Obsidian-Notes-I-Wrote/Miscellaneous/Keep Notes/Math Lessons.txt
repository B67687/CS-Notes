-- The Definition of a Function
- A FUNCTION is ONE INPUT, ONE OUTPUT. 
Therefore, when asking if y = x^2 - 1 is a function, one x gives one y.
Clearly any x will give one y.
But for y^2 = x - 1, one x will be related to 2 y's, because both the positive and negative value for y will be related to y^2, therefore one x gives two y's. 
Therefore not a function.
- Determining if relations are functions, check for x duplicate, then check for y duplicate. If no x duplicate then is a function, if x duplicate but same then still function, but if x duplicate but different then its NOT a function.
In summary: Check for even powers of y, or absolute values, or hidden even powers of y by factoring
